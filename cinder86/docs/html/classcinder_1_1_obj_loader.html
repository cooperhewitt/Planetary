Detailed Description

Loads Alias|Wavefront .OBJ file format.

Currently does not support anything but polygonal data
Example usage:

cinder::TriMesh myCube;
ObjLoader loader( loadFile( "myPath/cube.obj" ) );
loader.load( &myCube );
gl::draw( myCube );
